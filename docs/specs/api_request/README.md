## API request
 API is based on REST. It uses built-in HTTP features, like HTTP authentication and specific HTTP verbs.
 supports the following HTTP verbs:

**GET** - retrieves an item/items of resource type

**POST** - creates a new item of resource type

**PUT** - fully updates an existing item of resource type

**PATCH** - partially updates an existing item of resource type

**DELETE** - deletes an existing item of resource type

Each resource has at least the following endpoints:
Get by ID
Get list of (filter)
